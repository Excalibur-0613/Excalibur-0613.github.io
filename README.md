# Benjamin Russ | Robotics Portfolio

Welcome to my personal robotics portfolio built with GitHub Pages and Jekyll.

## About Me
I'm a robotics engineer with a strong interest in human-robot interaction, sim-to-real transfer, and real-time robotics systems. This site features key projects and personal testbeds that showcase my experience in simulation, hardware design, and system integration.

## Projects
- **Adaptive Multi-Robot Impedance Control**: Multi-agent coordination using Gazebo + ROS
- **Force-Sensitive Testbed**: Custom platform for analyzing real-time robot contact using FSRs
