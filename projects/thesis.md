# Adaptive Multi-Robot Impedance Control

A ROS-based multi-agent system exploring swarming and contact behavior via impedance control.

## Key Features
- Designed a force-aware control algorithm for multi-DOF robots
- Integrated Gazebo and ROS for real-time sim-to-real tests
- Physical robots equipped with custom sensors for contact feedback
