# Custom Force-Sensitive Mobile Robot Testbed (WIP)

A personal project to study force applications during mobile robot interactions.

## Goals
- Real-time force data capture using FSR sensors
- Stream data via microcontroller to ROS node
- Test dynamic interactions and apply real-time corrections

## Tools Used
- FSR sensor + ADC board
- Teensy microcontroller
- ROS2 + Python
- Mobile robot base

_This testbed is actively being developed and complements my research focus on physical interaction and contact response._
